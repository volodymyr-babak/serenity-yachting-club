# Serenity Yachting Club

Лендинг приватного клубу яхтових подорожей. Компанія орендує яхти по всьому світу,
розробляє авторські маршрути та організовує камерні групи. Офіс — у Києві.

**Живий сайт:** https://volodymyr-babak.github.io/serenity-yachting-club/

## Про сайт

- Односторінковий сайт (one-page) українською мовою.
- Стиль: сучасний «old money» — глибокий морський синій, слонова кістка, латунь,
  антиквенна типографіка (Cormorant Garamond + Jost).
- 6 готових турів з описами, маршрутами та орієнтовними цінами.
- Секції: про клуб, подорожі, як це працює, чому ми, галерея, відгук, контакти + форма запиту.
- Адаптивний, з м'якими анімаціями появи та мобільним меню.
- Форма бронювання працює через `mailto:` — жодного бекенду не потрібно.

## Технології

Статичний сайт без збірки: HTML + CSS + трохи ванільного JavaScript.

```
index.html            # розмітка та контент
payment-status.html   # сторінка повернення після оплати monobank
css/style.css         # дизайн-система і стилі
js/main.js            # навігація, анімації, форма, запуск оплати
js/config.js          # адреса платіжного Worker (paymentApi)
worker/               # Cloudflare Worker для оплати через monobank
```

## Оплата турів через monobank

На кожному турі є кнопка **«Забронювати»** — вона бере фіксовану **передоплату**
через інтернет-еквайринг monobank.

Оскільки GitHub Pages не має сервера, а токен мерчанта секретний, оплата працює
через маленький **Cloudflare Worker** (тека `worker/`), який тримає токен у себе,
створює рахунок у monobank і повертає посилання на сторінку оплати.

Поки `js/config.js` → `paymentApi` порожній, кнопки показують акуратне
повідомлення й ведуть до форми запиту. Щоб увімкнути реальну оплату —
розгорніть Worker за інструкцією в [`worker/README.md`](worker/README.md)
і впишіть його адресу в `js/config.js`.

## Локальний запуск

Достатньо відкрити `index.html` у браузері. Або підняти простий сервер:

```bash
python3 -m http.server 8000
# → http://localhost:8000
```
